## 누적 핵심 판단
### 브랜드/신뢰
- 공개 브랜드는 `EveryonePR / 모두의피알` 기준으로 유지한다.
- 공개면은 신뢰 -> 회원가입 -> 첫 주문 연결 구조를 먼저 본다.

### 주문/전환
- 로그인 이후 회원전용페이지 경험이 현재 가장 중요한 전환 병목이다.
- 매체 선택 / 가격 이해 / 주문 시작 동선은 단순화가 우선이다.

### 운영/구축
- preview / production stateful binding은 반드시 분리 유지한다.
- `member/admin/api/결제/도메인/Cloudflare binding`은 승인 없는 확장 금지다.

### 수익/오퍼
- 가격 할인보다 신뢰 구조와 재주문 용이성을 우선한다.
