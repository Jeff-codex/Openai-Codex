# 모두의피알 — NORTH STAR

## 1. 프로젝트 정의
모두의피알(EveryonePR)은 언론홍보가 필요한 중견기업, 스타트업, 브랜드, 광고대행사 등이
복잡한 중간 커뮤니케이션 없이 온라인에서 직접 매체를 선택하고 건별로 주문·결제·진행 확인까지 할 수 있게 만드는 언론송출 플랫폼이다.

## 2. 본질
이 프로젝트의 본질은 단순한 주문 툴이 아니다.
핵심은 **오프라인 영업 신뢰를 온라인 주문 시스템으로 전환하는 것**이다.

## 3. 핵심 타겟
- 브랜드 실무자
- 스타트업 대표 및 마케팅 담당자
- 중견기업 홍보/마케팅 담당자
- 광고대행사 AE 및 캠페인 운영 담당자

## 4. 사용자가 원하는 것
- 믿을 수 있는 매체 선택
- 불필요한 커뮤니케이션 없는 빠른 주문
- 명확한 가격과 진행 상태
- 실제 송출 가능성에 대한 신뢰
- 반복 발주가 쉬운 구조

## 5. 절대 목표
- 온라인에서 신뢰가 느껴지는 주문 구조 만들기
- 첫 주문 전환율 높이기
- 재주문 구조 만들기
- SEO/AEO/GEO 기반으로 언론홍보 수요를 지속 유입시키기
- 향후 플랫폼 확장 가능한 데이터/운영 구조 확보

## 6. North Star Metric
- 1차: 유효 방문자 대비 첫 주문 전환율
- 2차: 첫 주문 고객의 재주문율
- 3차: SEO/AEO/GEO 기반 유입 비중

## 7. 현재까지 완료된 기반
- `everyonepr.com` 공개면 전환 완료
- `review / member / admin` 호스트 전환 완료
- 공개 검색 구조와 가이드/인사이트 허브 구축 완료
- preview / production stateful binding 분리 완료
- 기본 보안 하드닝 완료

## 8. 현재 단계 핵심 과제
- 로그인 후 회원전용페이지의 신뢰감과 가독성 개선
- 주문/결제/매체선택 흐름의 명확성 강화
- 재주문을 쉽게 만드는 대시보드 구조 정비
- 검색 유입 이후 회원가입/첫 주문 전환 연결 최적화

## 9. 지금 하지 말아야 할 것
- 과도하게 무거운 엔터프라이즈 기능
- 이미 안정화된 도메인/보안/Cloudflare binding을 승인 없이 다시 여는 것
- 설명이 부족한 상태에서 매체/가격 구조를 복잡하게 만드는 것
- 실제 운영 프로세스와 분리된 UI만 먼저 다듬는 것

## 10. 제품 판단 원칙
1. 신뢰가 전환보다 선행한다.
2. 주문은 상담보다 쉬워야 한다.
3. 정보는 많기보다 명확해야 한다.
4. 오프라인 영업 자산은 온라인 전환 자산으로 번역되어야 한다.
5. 모든 페이지는 검색 유입과 전환 연결을 동시에 고려해야 한다.
